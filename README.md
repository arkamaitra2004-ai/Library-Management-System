# Library Management System

A Java and MySQL based console application for managing library books.

## Features
- Add Book
- Search Book
- Issue Book
- Return Book

## Technologies Used
- Java
- MySQL
- JDBC
- SQL

## Database Schema

Table: books

book_id | title | author | issued

## Concepts Applied
- Object-Oriented Programming
- JDBC Connectivity
- PreparedStatement
- SQL Queries
- CRUD Operations

## Future Enhancements
- Student Management
- Fine Calculation
- Due Date Tracking
- GUI using Java Swing
